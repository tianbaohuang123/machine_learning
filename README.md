## 机器学习、深度学习能做些什么

* 图像：图像识别，无人驾驶
* 医疗：识别患者CT图等，
* 图片艺术化：替代PS
* 人脸识别
* 新闻一“小”一“同”：南方都市报的“小南”，广州日报的“阿同”机器人
* 信贷需求预测
* 店铺销量预测
